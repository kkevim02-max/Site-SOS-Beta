# Alterações desta versão

- Conteúdo de **Tempos Litúrgicos** incorporado à página **Liturgia**.
- Link “Tempos” removido de todos os menus.
- Página antiga de tempos mantida apenas como redirecionamento, evitando links quebrados.
- Identidade católica reforçada com referências a Cristo, Eucaristia, Igreja, Maria, missão e versículos bíblicos.
- Página inicial recebeu uma área “Nossa fé no centro”.
- Página “Quem somos” recebeu uma seção de espiritualidade.
- Página de contato foi corrigida e reorganizada.
- Menu para celular virou um painel lateral com fundo escurecido e fechamento por toque, link ou tecla Esc.
- Títulos, botões, cartões, espaçamentos e navegação foram ajustados para telas pequenas.
- Navegação interna da Liturgia ficou horizontal e fixa no celular.
- PDF da folha foi renomeado para um nome seguro para GitHub Pages: `15-domingo-tempo-comum-sos.pdf`.
