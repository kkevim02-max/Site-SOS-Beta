# Portal S.O.S — pronto para GitHub Pages

Site estático completo, sem login e sem banco de dados.

## Conteúdo incluído
- 16 aulas de violão baseadas na apostila do S.O.S., com imagens do próprio material
- 18 aulas de canto
- 18 aulas de teclado
- 16 aulas de contrabaixo
- Formação litúrgica
- Tempos litúrgicos
- Sistema simples de folhas e apostilas por arquivos de dados
- Layout responsivo e animações leves

Leia `COMO-EDITAR-E-PUBLICAR.md`.

- Apoios visuais em todos os instrumentos
- Imagens ampliáveis ao clicar


## Animações

O site possui animações automáticas ao rolar. Novos cartões, aulas, folhas e blocos recebem movimento sem precisar editar cada página. O sistema está em `assets/js/script.js` e `assets/css/style.css`.
